# ASE-project

## Database
You can connect to the database with the following command.
```
psql -U postgres
```
Here some useful `psql` commands.
```
\?                 # help
\l                 # list databases
\c <db>            # connect to a database
\conninfo          # show info about current database connetion
\dt                # list tables
\q                 # quit
```
